## 1。GPT-4

「 GPT-4 」是OpenAI开发的大型语言模型。

在多模式中,您现在可以接受文本和图像提示。令牌的最大数量已从4K增加到32k。推理能力也在显着提高。

当前「[ChatGPT Pro](https://chat.openai.com/chat)」 (付费版本)有限制,但重量列表的注册人除外「[OpenAI API](https://openai.com/blog/openai-api)它也已开始与」一起使用。

## 2。传电

「 PaLM 」是Google开发的大型语言模型。

我们计划在不久的将来发布「 PaLM API 」的重量列表,可用于「 PaLM 」。 「PaLM API 」是一个API,可轻松使用Google的大型语言模型,使您可以访问针对内容生成和聊天而优化的交互式模型,以及为摘要和分类而优化的通用模型。
