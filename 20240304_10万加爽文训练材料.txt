减掉216斤的亚洲首富之子，又胖回了400斤...

果然一百岁不死都有新鲜事。

自贾玲爆瘦100斤刷爆全网后，最近又一个减肥奇闻炸上热搜——

亚洲首富之子阿南特·安巴尼 18个月减掉216斤。

Image

什么概念？

相当于减掉了曾经的一个贾玲......

Image

这哪是瘦身啊？分明是大变活人！

细看之下，瘦下来的阿南特眉宇间甚至有点像还在里面踩缝纫机那位。

Image

消息一出，全网又惊又酸：短短18个月，我们就失去了1/2个世界首富（之子）。

可背后的真相，震碎了居里的三观。

据外媒报道，他的确是瘦了200斤。

但他现在又胖了200斤！！

在去年11月的活动图中，阿南德长这样。

Image

旁边站着的，正是即将在7月喜结连理的未婚妻拉迪卡·麦钱特。

Image

说实话，他减200斤我拍手叫好，但他是亚洲首富之子我彻夜难眠。

阿南特是穆克什·安巴尼和夫人妮塔·安巴尼最小的儿子。

Image

据《福布斯》富豪榜，安巴尼身价 1,120 亿美元（约合8058亿人民币），手握印度最大的私人联合企业——信实工业集团。

富可敌国的安巴尼，出了名的宠妻狂魔。

Image

最出圈的战绩莫过于因老婆一句“想看到孟买的全景”，他便大手一挥豪掷10亿美元建造了这栋名叫安迪利亚的27楼公寓。

能与之一较高下的私宅，只有白金汉宫。

Image

俗话说，幺儿幺心肝。

作为两人最小的儿子，阿南特自然万千宠爱于一身。

可这宠爱，400多斤重。

Image

安巴尼一家从来都是身光颈靓，阿南特的哥姐更是才貌双全。

但镜头中的阿南特，危危乎若巨肉山欲倾，颤颤然如不倒翁坠地。

一家合照，他光靠肚子就能占1/2，配上这个清澈的眼神，像动画角色。

Image

家丑不可外扬，几年前其母才透露，原来阿南特自幼患有哮喘，长期服用类固醇，导致一度重达408斤

2016年，父母找到全印度最著名、私教要价1.3 w 的健身教练畅拿。

Image

减肥，无外乎一句“管住嘴，迈开腿”。

可少爷吃惯了山珍海味，怎咽得下粗茶淡饭？

畅拿主张低脂肪、低碳水化合物、高蛋白、零糖饮食，再请来顶级大厨烹调，花样百出，再配上新鲜水果和蔬菜。

让我想起了《家有儿女》里，孩子们想吃忆苦餐，姥姥用鸡汤熬一根野菜的名场面。

Image

运动上，严格管控。

少爷每天运动5-6个小时，包括有氧、瑜伽、力量训练、灵活性练习，外加步行21公里。

但这对普通人毫无参考价值，因为咱下班后睡觉时间都没6小时。

短短18个月，少爷就只剩下了1/2。

Image

下颌线出来了，精气神也好了，太后在一旁喜笑颜开，很欣慰。

Image

但帅不够三秒，瘦不够一年。

因为药物影响，加上疫情松懈，阿南特迅速反弹。

一胖又回到了解放前，有钱人果然都是瘦着玩的......

Image

我就纳闷了，人都胖回去了，为啥热搜又爆了？

原来是为了给他的世纪婚礼造势。

Image

3月1日-3日，安巴尼将为儿子和准儿媳举办一场为期三天的婚前派对。

名单、流程一曝光，“元妃省亲”都甘拜下风。

表演嘉宾，蕾哈娜。

Image

是的，整个乐坛、千万粉丝抬都抬不出的美妆富婆，已经到达现场了。

世界上只有一种超能力，就是钞能力。

Image

他们家请明星唱歌，就像是农村请剧团唱戏一般稀疏平常。

长子结婚请酷玩，二姐结婚请碧昂丝。

当年请碧局甩了1.3亿人民币，连演出服都给她镶满850 颗红宝石和祖母绿。

Image

宾客更是豪华：比尔·盖茨、扎克伯格、伊万卡·特朗普......

不说是吃席，我还以为是什么国际财富论坛。

派对将在一个3000英亩的花园中举行，里面种有数万棵花树，育有2000多只被解救的野生动物：豹子、美洲狮、大象等。

安巴尼夫妇准备了2500道菜肴，确保宾客绝不会吃到相同的美食。

最夸张的是请柬中有一叠9页厚的着装要求，一句话：怎么壕怎么来。

Image

著名婚礼编辑阿斯特丽德·乔斯总结：史无前例。

没办法，谁让这个准儿媳深得太后欢心呢。

Image

未来三少奶拉迪卡，制药大亨之女，出落得亭亭玉立，眉目如画。

无论是多么繁复浓艳、珠光宝气的传统服饰，都能给她演绎得山清水秀。

Image

时髦利落的礼服，她也驾驭得云淡风轻。

Image

拉迪卡和阿南特乃青梅竹马，大学后互生情愫。

Image

你问我什么是真爱？

当她陪着一个男人从400斤-200斤-400斤，仍满眼小星星，我觉得这就是真爱。

当然，如果一个人家里8000多个亿，我也不会在意其外表。

ImageImage

所以，很多年前，安巴尼夫妇早把拉迪卡当作了家庭一员。

另外，拉迪卡是一位训练有素的印度古典舞者，而太后妮塔自幼习婆罗多舞，被称为“为神跳舞”的神女。

Image

太后不仅舞好，而且瘾大。

去年3月她艺术中心开幕，请来了 Gigi、赞达亚众星，谁料开幕演出就是妮塔本人。

有一种公司开张，舞狮的是董事长的感觉。

Image

可想而知，当太后看到一个和自己一样能作惊鸿舞的女子，会有多心花怒放。

而安巴尼家的男人都有一个优良传统：我可以不穿名牌，但我老婆的包必须要买最贵的。

安巴尼老爷子就做了很好的榜样。

Image

老婆想要车车，他就买下一部镶满了钻石的奔驰。

居里20多年的密集恐惧症，突然就好啦。

Image

老婆手机坏坏，他就下单最新款的苹果，然后在后面镶一颗比胆结石还要大的宝石。

太后表示，挺好的，就是有点重啦。

Image

但长年累月的“负重”，妮塔的硬拉已经是大神级别。

光是这些比节能灯泡都要大的宝石首饰，就很考脊椎的稳定度。

Image

更别提富婆们配货都要等半年的铂金包，在太后面前，那就是一个菜篮子。

Image

在太后手把手的点拨之下，准媳妇拉迪卡也肉眼可见的珠辉玉丽。

看看这满眼的宠溺。

Image

对小儿子这场婚礼，安巴尼夫妇不惜代价，穷奢极侈。

安巴尼甚至捐款重修了14座寺庙，作为儿子儿媳签字的宝地。

Image

这不仅是展现对儿子儿媳的宠爱，更是向世界富豪炫耀家族的财力。

只能说，抖音的豪门短剧，想象力还是不够啊。

有钱人的生活我们无法想象，但居里和网友只有一个要求：

麻烦安巴尼老爷子让蕾哈娜赶紧把专辑发了！
