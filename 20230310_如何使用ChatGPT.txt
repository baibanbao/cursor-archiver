如何使用ChatGPT

简单地说有两个渠道，第一，自己搞定，第二交给第三方，第三先买个API用着再说

方案一

注册OpenAI

准备科学上网

准备一个海外手机号来接收验证短信，可以用海外虚拟号码https://sms-activate.org/

注册完成后，进入API页面 创建Key，然后就可以使用了。

这个方案目前可行，是因为OpenAI给每个新用户提供了18美金的免费额度。但是一旦不再提供，就会面临充值的问题。目前OpenAI不接受中国信用卡，因此还必须准备一个海外信用卡。

不过万能的某宝可以买到借记卡并且可以充值（卡的所有权不在你手上，所以还是有风险，建议每次充值不要太多）

方案二：

通过第三方接口访问

如果你搞不定海外手机号和信用卡，或者自己不想架设代理，那么可以考虑用像API2D这样的第三方代理API。

方案2的链接：https://api2d.com/r/186124

方案2，也需要注册有GitHub帐号，并且要有一定的技术能力才能使用，请研究清楚了再购买。

主要的优点：

基本兼容原有接口，只需要改下 API endpoint 和 Key

支持国内卡充值，开发者可以让用户自己自行购买点数并创建Forward Key

接口国内直接可以访问，无需架设代理

推荐可以获得点数，这里是我的推荐链接

缺点：

不支持 stream 参数，因此只能一次性返回内容

不支持微信充值，价格比官方略高

方案三

如果嫌以上办法太麻烦，先买个openAI的API用着再说，可以到这里这里购买：https://openaikey.xyz，还聚合了许多能够使用 API Key 的工具。

价格9.9元人民币一个，内带5美元，使用期限3个月。

不过买了API，一般也要用科学上网才能用。
